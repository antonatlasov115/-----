# 🎮 Быстрый старт - Онлайн мультиплеер Сонор

## Для Windows (самый простой способ)

1. Дважды кликните на файл `start-multiplayer.bat`
2. Подождите, пока откроются два окна (сервер и клиент)
3. Откройте браузер на `http://localhost:5173`

## Для Mac/Linux

```bash
chmod +x start-multiplayer.sh
./start-multiplayer.sh
```

## Ручной запуск

### Шаг 1: Запустите сервер

```bash
cd server
npm install
npm start
```

## Как играть онлайн

### Игрок 1 (создает комнату):

1. Нажмите **"Online Multiplayer"**
2. Нажмите **"Create New Room"**
3. Скопируйте **Room ID** (например: `ABC123`)
4. Нажмите **"Connect to Server"**
5. Дождитесь статуса 🟢 **Connected**
6. Отправьте Room ID второму игроку

### Игрок 2 (присоединяется):

1. Откройте игру в другом браузере/вкладке
2. Нажмите **"Online Multiplayer"**
3. Введите **Room ID** от первого игрока
4. Нажмите **"Join Room"**
5. Дождитесь подключения

### Начало игры:

- Игроки ходят по очереди
- Ходы синхронизируются автоматически
- Счет обновляется в реальном времени

## Проверка работы сервера

Откройте в браузере:
- Health check: http://localhost:3001/health
- Список комнат: http://localhost:3001/rooms

## Troubleshooting

### ❌ Порт 3001 занят

```bash
# Windows
netstat -ano | findstr :3001
taskkill /PID <PID> /F

# Mac/Linux
lsof -ti:3001 | xargs kill -9
```

Или запустите на другом порту:
```bash
PORT=3002 npm start
```

### ❌ Клиент не подключается

1. Убедитесь, что сервер запущен
2. Проверьте консоль браузера (F12)
3. Проверьте консоль сервера
4. Попробуйте перезапустить оба процесса

### ❌ Игра рассинхронизировалась

1. Оба игрока нажимают **"Disconnect"**
2. Перезапустите сервер
3. Создайте новую комнату

## Особенности

✅ **Автоматическая синхронизация** - Все ходы передаются мгновенно
✅ **Валидация ходов** - Невалидные ходы отклоняются
✅ **Обработка отключений** - Игра корректно обрабатывает разрывы соединения
✅ **Множество комнат** - Можно создать несколько игр одновременно
✅ **Логирование** - Все события логируются для отладки
