# 📱 Мобильные возможности Sonor

## Новые функции оптимизации для мобильных устройств

### 🔍 Управление зумом

#### 1. **Pinch-to-Zoom (жест двумя пальцами)**
- Сведите/разведите два пальца на экране для зума
- Плавное масштабирование от 0.5x до 5x
- Зум центрируется между пальцами

#### 2. **Двойной тап**
- Быстро коснитесь экрана дважды для зума 2x
- Повторный двойной тап возвращает зум к 1x
- Зум центрируется на точке касания

#### 3. **Кнопки зума**
- **+** — увеличить (1.3x за клик)
- **−** — уменьшить (0.77x за клик)
- **⊙** — сбросить зум и позицию

#### 4. **Колесико мыши** (на планшетах с мышью)
- Прокрутка вверх — увеличение
- Прокрутка вниз — уменьшение
- Зум центрируется на курсоре

### 🎮 Улучшенное управление

#### Перемещение доски
- **Перетаскивание одним пальцем** (когда нет активной фишки)
- Плавная прокрутка по всей доске
- Автоматическая блокировка при выборе фишки

#### Управление фишками
- **Тап на фишку** — выбрать
- **Перемещение пальца** — двигать фантомную фишку
- **Кнопка "✓ Ход"** — подтвердить ход
- **Кнопка "✕ Отмена"** — отменить выбор

### 📐 Адаптивная верстка

#### Viewport оптимизация
- Использование `dvh` (dynamic viewport height) для корректного отображения на iOS
- Автоматическая адаптация к изменению ориентации
- Минимальные отступы для максимальной игровой области

#### Responsive breakpoints
- **Desktop** (>768px): боковая панель слева
- **Mobile** (≤768px): выдвижная панель сверху
- **Landscape**: оптимизированная высота панели

### 🚀 Производительность

- Debounced resize для плавности
- Touch action optimization
- Предотвращение случайного зума браузера
- Отключение highlight при касании

### 🎨 UX улучшения

- Большие кнопки (50x50px) для удобного нажатия
- Градиентные кнопки в стиле игры
- Тени для визуальной глубины
- Плавные анимации (0.3s transitions)

## Технические детали

### Диапазон зума
- **Минимум**: 0.5x (50%)
- **Максимум**: 5x (500%)
- **По умолчанию**: 1x (100%)

### Touch события
- `touchmove` — перемещение фишки / pinch zoom
- `touchend` — завершение жеста
- `tap` — двойной тап для зума

### Совместимость
- ✅ iOS Safari
- ✅ Android Chrome
- ✅ Desktop browsers
- ✅ Планшеты с touch screen

## Советы по использованию

1. **Для точных ходов**: увеличьте зум до 2-3x
2. **Для обзора**: используйте зум 0.5-1x
3. **Быстрый зум**: двойной тап на нужной области
4. **Навигация**: перетаскивайте доску одним пальцем
5. **Сброс вида**: нажмите кнопку ⊙

## Известные ограничения

- Pinch-zoom работает только при двух пальцах
- Перетаскивание доски блокируется при активной фишке
- На очень старых устройствах возможны задержки при зуме
