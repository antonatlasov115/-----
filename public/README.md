# Public Assets

Эта директория содержит статические файлы, которые копируются в корень сборки.

## Файлы

### board.png
Изображение игровой доски для фона.

**Источник:** `graphics/доска.png`

**Использование:** Автоматически загружается в компоненте GameBoard как фон canvas.

**Размеры:** Масштабируется под размеры доски из `gameConfig.ts`:
- Ширина: 1200px
- Высота: 800px

## Добавление новых ресурсов

1. Поместите файл в директорию `public/`
2. Файл будет доступен по пути `/filename.ext` в приложении
3. Для продакшн сборки файлы автоматически копируются в `dist/`

## Примечание

Vite автоматически копирует содержимое `public/` в корень `dist/` при сборке.
